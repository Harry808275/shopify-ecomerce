## Step 1: Install the App

1. **Click on the installation link** you just provided, OR
2. Copy and paste it into your browser
3. This will take you to your Shopify admin to install the app
4. Click **"Install app"** when prompted

## Step 2: Enable Storefront API (If Not Done Yet)

1. Go back to your **Shopify Partners** dashboard
2. Navigate to: **API access** page (for your "abaya" app)
3. In the **"Storefront API"** section, click **"Enable Storefront API"**
4. Configure the scopes:
   - `unauthenticated_read_product_listings`
   - `unauthenticated_read_collection_listings`

## Step 3: Get Your Storefront API Access Token

You have two options to find the token:

### Option 1: Dev Dashboard (Recommended)
1. Click on **"Dev Dashboard"** link in the blue banner at the top of the page
2. Look for **"Storefront API access token"** or **"API credentials"** section
3. Copy the token (it will look like: `shpat_xxxxxxxxxxxxx` or `shpca_xxxxxxxxxxxxx`)

### Option 2: Shopify Admin
1. Go to your **Shopify Admin**: `gando2.myshopify.com/admin`
2. Navigate to: **Settings** → **Apps and sales channels**
3. Find your **"abaya"** app in the list (or go to **"Develop apps"** if not visible)
4. Click on it
5. Look for **"Storefront API access token"** section
6. Click **"Reveal token"** or **"Copy token"**
7. Copy the token
