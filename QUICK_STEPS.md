# Quick Steps - Get Your Credentials

## Right Now - On Your Current Page:

### 1. First, Set Distribution Method
- Click **"Select distribution method"** (in the Storefront API section)
- OR go to **"Distribution"** in the left sidebar
- Choose **"Custom app"**
- Save

### 2. Enable Storefront API
- On the same page you're on now (API access)
- Find the **"Storefront API"** section
- Click **"Enable Storefront API"** button

### 3. Get the Access Token
After enabling, you need to go to your **Shopify Admin** (not Partners):
- Go to: `yourstorename.myshopify.com/admin`
- Settings → Apps and sales channels → Develop apps
- Click on "abaya" app
- Install/Configure the app
- Copy the **Storefront API access token**

### 4. Get Your Store Link
- Your store link is: `yourstorename.myshopify.com`
- Check the URL when you're in your Shopify admin
- Or go to Settings → Domains
